Amazon Prime Data Analysis | Power BI Project
Overview
This project presents a Power BI-based data analytics solution focused on Amazon and Amazon Prime datasets. It leverages a structured semantic model to deliver insights through interactive dashboards, trend analysis, and time-based reporting.
🎯 Objectives
Analyze Amazon Prime and Amazon datasets
Perform time-based analysis using a calendar table
Build a scalable and structured data model
Generate actionable business insights
Data Model
The project consists of the following tables:
Amazon Prime Data – Contains Prime-related content or service data
Amazon_Data – General Amazon dataset (products/transactions)
Calendar Table – Date dimension for time intelligence
🔗 Model Design
Star-schema inspired structure
Calendar table connected for time-based filtering
Supports aggregation, filtering, and comparisons
📊 Features
📅 Time-based analysis (monthly, yearly trends)
📈 Interactive dashboards
🔍 Data filtering and slicing
⚡ Optimized model with compressed storage
🛠️ Tools & Technologies
Power BI (Desktop / Service)
Data Modeling
DAX (for calculations, if used)
📈 Use Cases
Analyze content trends on Amazon Prime
Track performance over time
Compare categories and datasets
Generate business insights
